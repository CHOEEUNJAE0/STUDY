document.getElementById("아이디 속성 값")

HTML 태그 요소의 아이디 속성 값을 지정하여 요소 정보를 가져온다.
★주의 : 아이디 속성의 값은 중복 되어서는 안된다.
중복 되는 경우 원하는 요소를 찾을 수 없다.

document.getElementsByName("네임 속성 값")

HTML 태그 요소의 네임 속성의 값을 지정하여 요소 정보를 가져온다 네임 속성의 값은 중복 할 수 있으며, 중복 되는 모든 요소 정보를 찾는다.

document.getElementByTagName("태그 속성 값")

HTML 태그명을 지정하여 해당 태그의 요소 정보를 가져온다.
동일한 태그명을 사용하는 모든 요소 정보를 찾는다.

document.getElementsByClassName("클래스 속성 값")

HTML 태그 요소의 클래스 속성의 값을 지정하여 요소 정보를 가져온다.
클래스 속성의 값은 중복 할 수 있으며, 중복 되는 모든 요소 정보를 찾는다.

document.querySelector("CSS 선택자")

HTML 태그 요소 정보를 찾을 때 CSS 선택자를 사용하는 방식으로 요소 정보를 찾는다.
하나의 요소만을 찾아준다.

document.selectAll("CSS 선택자")

HTML 태그 요소 정보를 찾을 때 CSS 선택자를 사용하는 방식으로 요소 정보를 찾는다.
모든 요소를 찾아준다.
